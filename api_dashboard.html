Class dashboard¶

Import strymmap as:

from strym import dashboard

helps in collecting metadata from within a CAN CSV folder and print interesting aspects of the collection.

class strym.dashboard(directory='./', verbose=False, start=None, end=None, **kwargs)¶

dashboard works within the strym package to collect metadata files from within a folder and print interesting aspects of the collection
